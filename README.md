# Project Description
The goal of this project is to create a Blog web application using Node.js, Express.js, and EJS. 
The application will allow users to create and view blog posts. 
Posts will not persist between sessions as no database. 
Styling will be an important aspect of this project to ensure a good user experience.

# Deliverables
	• One Node project for the function of the website
	• Include at least one EJS file for the structure of the website
	• Include at least one CSS file for the styling of the website

# Features
	• Post creation: users should be able to create new posts.
	• Post Viewing: Home page should allow user to view all posts.
	• Post Update/Delete: Users should be able to edit and delete posts as needed.
Styling: Responsive on both desktop and mobile devices.
